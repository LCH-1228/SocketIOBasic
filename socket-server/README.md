# Socket.IO Learning Server

iOS 앱 프로젝트에서 `Socket.IO`를 학습하기 위해 작성한 Node.js 서버입니다.  
서버는 기본적으로 `http://localhost:8080`에서 실행됩니다.

## 실행방법
```bash
npm install
npm start
```

## 구조
```
socket-server
├── .gitignore
├── package.json
├── README.md
└── server.js
```

## 실행환경
- Node.js 25.1.0  
- Socket.IO 4.8.1

해당 환경에서 서버 정상 동작 확인했습니다.
